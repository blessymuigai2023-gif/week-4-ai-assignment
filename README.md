## What is Data?
Data is raw, unorganized facts, figures, symbols, or observations that lack inherent meaning until processed.
## Explain the difference between:
- **Structured Data**- This is data that is organized into a fixed, predictable format that a computer can easily read and search, for example: *Bank Statements* and *phone numbers*.
- **Unstructured Data**- Unstructured data is data that doesn't have a predefined format or organization. for example: *photos* and *audio*.
## What is a schema?
This is the blueprint or master plan that defines how data is organized, structured, and related within a database.
